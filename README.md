Sandbox for armv7 based Android 
====================================================
Currently supported : 
* libvnc 0.9.9
* ZeroMQ 4.0.4
* JPEG
* JPEG-turbo
* libpng
* sth....

(formerly of Olivetti Research Ltd / AT&T Labs Cambridge).
-> http://www.realvnc.com/docs/rfbproto.pdf
